# 驾校管理系统 java1009

驾校管理系统是一款基于SpringBoot和Vue的前后端分离的软件应用，致力于提高驾校运营效率和管理水平。

## 技术栈

- 后端：SpringBoot
- 前端：Vue
- 数据库：MySQL
- 开发工具：Idea，vscode
- 依赖框架：mybatis-plus，element

## 功能模块

### 系统角色

- 普通用户
- 教练
- 超级管理员

### 功能介绍

- **驾校车辆管理**：用于管理驾校的车辆信息。
- **预约考试管理**：协助用户进行考试预约，并管理预约信息。
- **学员成绩管理**：记录和管理学员的考试成绩。
- **驾校知识管理**：管理驾校相关的理论知识内容。
- **考试管理**：负责考试流程和内容的设置与管理。
- **我的收藏管理**：允许用户收藏常用功能或信息。
- **个人中心**：提供个人资料、设置等个性化管理功能。
- **个人后台**：用户可以查看和管理与自己相关的后台数据。

## 运行环境

- Java 8 或更高版本
- MySQL 5.7 或更高版本
